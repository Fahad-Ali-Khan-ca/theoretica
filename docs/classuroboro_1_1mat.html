inline

Independent at() function.

Access element at i and j index, where i is always the index on rows and j is always the index on columns.

This function is used inside of the library to access matrix elements independently from the specific setup of storage or notation (regardless of UROBORO_MATRIX_LEXIC and UROBORO_ROW_FIRST)
