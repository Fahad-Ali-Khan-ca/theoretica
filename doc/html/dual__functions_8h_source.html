8// Dual numbers can easily be used for
9// automatic differentiation, as they behave
10// under addition, multiplication and divisions
11// as the first derivative.
12
13// Normal operations can be performed and the result
14// will have a real part equal to the function evaluated
15// for the given argument and a "dual" part equal
16// to the first derivative evaluated for the given argument.
